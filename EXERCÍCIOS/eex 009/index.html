Ordered lists (listas ordenadas)

Listas oredenadas ou ordered lists, são listas como a lista de supermercado, lista de aprovação, listas que tem uma ordem lógica a se seguir

  1. começo
  2. Meio
  3. Fim

Unordered lists (listas não ordenadas)

listas não ordenadas diferentemente das listas ordenadas não seguem uma lógica

  • Fim
  • Meio
  • começo
