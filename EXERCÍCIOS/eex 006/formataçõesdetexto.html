Principais Formatações de Texto

Negrito/Destaque

Usando a tag B obtemos o mesmo resultado, porém esta tag não tem valor semântico

Mas para obtermos o mesmo resultado com valor semântico utilizamos a tag strong.

Itálico/ênfase

nas versões anteriores era utilizado a tag I, porém conforme o exemplo anterior obtemos o mesmo resultado porém saem valor semântico

Na versão atual utiliza-se a tag EM para colocar tal citação em negrito ou itálico.

Texto Marcado

Para este tipo de texto temos a tag MARK onde destaca-se o exto desejado.
