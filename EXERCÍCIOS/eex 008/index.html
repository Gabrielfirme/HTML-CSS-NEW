Principais formatações

Negrito/ Destaque

Para colocar termos em Negrito , eu uso a tag STRONG ppois as outras para este fim já são obsoletas.

Itálico/ Ênfase

Para colocar certo termo em ênfase ou itálico simplesmente usamos a tag EM para colocar em Ênfase de forma semântica

Texto marcado

Para criar textos marcados é simples basta usar a tag MARK

assim também

Texto deletado

Usa-se a tag DEL para indicar que aquele texto em si foi deletado ou não deve ser lido.

Texto Grande e texto pequeno

para colocarmos texto grande é simples, tão qual colocar textos pequenos

Texto Inserido

O texto inserido é a forma semântica da antiga Tag U deixando assim sublinado a parte mais importante do conteúdo, para isso usa-se a tag INS

Texto Sobreescrito

para inserir coisas do tipo x 20 +3 é necessario usar a tag SUP.

texto subscrito

para escrever um texto subscrito do tipo H2O é necessário usar a tag Sub
